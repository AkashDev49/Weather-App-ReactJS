# 🌦️ Weather App

A modern and responsive **Weather Application** built with **React.js** that allows users to search for the current weather conditions of any city around the world. The application fetches real-time weather data using a Weather API and displays important information such as temperature, humidity, wind speed, weather conditions, and more.

## 🚀 Features

* 🌍 Search weather for any city in the world
* 🌡️ Real-time temperature updates
* 💧 Humidity information
* 💨 Wind speed details
* ☁️ Weather condition with icons
* 📱 Fully responsive design
* ⚡ Fast and smooth user experience
* 🔄 Dynamic API integration

## 🛠️ Tech Stack

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3
* Weather API
* Axios / Fetch API (depending on your implementation)

## 📚 What I Learned

This project helped me strengthen my understanding of React fundamentals and API integration. Through this project, I learned:

* How to fetch real-time data from a REST API in React.
* How to use the **useEffect() Hook** for handling side effects such as API calls.
* How to manage application state using the **useState() Hook**.
* Understanding component-based architecture in React.
* Handling asynchronous operations using **async/await**.
* Error handling while making API requests.
* Rendering dynamic data received from APIs.
* Managing user input using controlled components.
* Creating a responsive UI for different screen sizes.
* Organizing React projects with reusable components.
